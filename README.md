# PPL + Upper/Lower Workout Website

This is a static website for the workout plan from the Google Sheet.

## Files
- `index.html` — website structure
- `styles.css` — design and responsive layout
- `script.js` — workout data, filters, checklist progress, and local notes

## How to use
Open `index.html` in a browser.

## How to edit exercises
Open `script.js` and edit the `workouts` array.

## How to publish
Upload the folder to GitHub Pages, Netlify, Vercel, or any static hosting service.
